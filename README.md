# 🚀 ERP API Automation Framework (REST Assured)

## 📌 Overview
This project is an API automation framework built using **REST Assured + Java + TestNG** for testing ERP system APIs.  
It supports scalable, maintainable, and reusable API test design using industry best practices.

---

## ⚙️ Tech Stack
- Java
- REST Assured
- TestNG
- Maven
- Jackson (POJO serialization)
- Eclipse IDE

---

## 📂 Project Structure
src/test/java
 ├── base        → Base setup (base URI, config)
 ├── tests       → API test cases
 ├── pojo        → Request/Response models
 ├── specs       → Request & Response specifications
 └── utils       → Test data utilities

---

## 🔐 Features
- REST API automation using REST Assured
- POJO-based request payload handling
- Request & Response specifications
- Reusable test structure
- Validation of status codes and responses
